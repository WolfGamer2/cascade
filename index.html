About the YSWS

Make a CSS animation! Use only CSS to create a cool animation and get some art supplies.

How to Participate

  1. Design a creative CSS animation (check criteria for what counts as creative)
  2. Make your CSS animation in CodePen!
  3. Submit your CodePen and code to the submission form here by the deadline
  4. If you're approved, we'll be in touch to arrange the shipping of your choice between art supplies and art software.

Judging Criteria

  • Creativity: How creative the animation is.
  • Visual Appeal: The animation should look good.
  • Elements: Must use 2+ elements like text, shapes, and colors.
  • Uniqueness: Must be unique. (Cannot do a project similar to someone else's).
